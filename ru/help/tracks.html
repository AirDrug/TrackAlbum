Справка / Работа с треками


Работа с треками происходит на вкладке GPS треки, подвкладка Треки:

Области на вкладке:
  • 1. Список треков (имя трека и его длина)
  • 2. Карта трека
  • 3. Таблица со свойствами трека
  • 4. График высоты трека

Список треков

В списке показываются все треки проекта.

Что-бы добавить трек в проект нужно либо скачать его с GPS либо импортировать из файла.

Список можно сортировать по имени трека и по его длине. Имя треку рекомендуется давать по шаблону ГОД-МЕС-ДЕНЬ-СОБЫТИЕ-МЕСТО.

Для работы с треком кликните по нему левой кнопкой мыши. Для вызова контекстного меню кликните правой кнопкой мыши.
Иногда может понадобится выбрать 2 и более треков, удерживайте кнопку Ctrl для этого.

Карта

На карте (2) показываются выбранные треки.
  • Для показа нескольких треков выберите нужные треки, удерживая клавишу Ctrl.
  • Трек раскрашивается в зависимости от типа выбранной раскраски.
    Можно раскрасить часть трека красным цветом, для этого выделите нужный участок на цветной шкале, кликнув начало участка левой кнопокй мыши и удерживая ее переместите мышь до нужной точки:
  • Трек можно двигать мышкой, зажав левую кнопку.
  • Изменять масштаб можно, зажав правую кнопку и двигая мышь вверх/вниз или нажимая кнопки с изображением лупы.

График высоты

  • График высоты (3) показывает высоту трека в определенный момент времени.
  • График также раскрашивается в зависимости от типа выбранной раскраски.
  • Двигая мышью по графику высоты можно слева графика посмотреть параметры точки трека: Время, Высоту, Скорость, Скороподъемность в данной точке.
  • Так-же на карте трека показывается расположение данной точки.
  • График можно растягивать по горизонтали кнопками с изображением лупы и двигать по горизонтали, зажав правую кнопку мыши.

Импорт треков

Программа умеет читать самые популярные форматы: IGC, OZI, TRK, GPX, WPT.

Для импорта трека:

  • Выберите меню Файл/Импортировать трек
  • Выберите файлы треков (можно выбрать несколько файлов за один раз, придерживая клавишу Ctrl)
  • Нажмите кнопку 'ОК'

Анализ трека и части трека

Выделите нужную часть трека на графике, зажав левую кнопку мыши.
Свойства выделенного участка будут показываться в таблице свойств трека (4).

Если нет выделенного участка трека, то показываются свойства всего трека.

Просмотр точек трека

  • Выберите трека в списке треков
  • Правой кнопкой мыши вызовите контекстное меню и выберите пункт Показать точки

Изменение треков

С треком возможны следующие операции:

Разбивка трека на 2 трека

  • Выберите нужный трек в списке
  • Выполните двойной щелчок или правой кнопкой мыши вызовите контекстное меню и выберите пункт Показать точки
  • Выберите точку с которой начнется второй трек
  • Правой кнопкой мыши вызовите контекстное меню и выберите пункт Разбить трек
  • В появившемся окошке наберите имя нового трека, нажмите 'Ok'

Объединение треков

  • Выберите 2 трека в списке треков, удерживая клавишу Ctrl
  • Правой кнопкой мыши вызовите контекстное меню и выберите пункт Объединить треки
