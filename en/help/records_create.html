Help / Working with log book / Creating and editing a record


To create record switch to Log Book sheet and press 'Add record' button, record editing form will pop up.

Depending on project type the form contains different sets of fields.
Values in these fields are used to calculate statistics.

Pay attention to text fields with drop down list, it contains a list of all values entered earlier. Also start typing and the program tries to autocomplete values.


For flight log enter:

  • Date
    Flight date, default value is today, press arrow on the right to open calendar.

  • Glider
    The name of you aircraft. Field has autocompletion.

  • Ground handling
    Set this if you did not fly but had exercises with wing on the ground.
    FIelds 'Flight type', 'Launch method' will be set automatically, some fields will be disabled.

  • Flight type
    For example: x-country, dynamic, thermodynamic...

  • Flight count
    If you had several flight in a day you can group them in one record.

  • Total duration
    Total duration of all flights in the group.

  • Best duration
    Enter duration of the longest flight in the group.

  • Location
    The name of flight site.

  • Launch method
    For example: 'static winch', 'aircraft towing' or left empty if launched from legs.

  • Mountains
    Set this if it was mountain flight.

  • Event
    Arbitrary event name, for example 'X-Country course','WolrdCup 2007'.

  • Length
    Enter for x-country flights.

  • Comments
    Your notes for the flight, comments are shown in record details on 'Log book' sheet.

  • Linked pictures
    Use 'Add picture','Delete' to attach and detach pictures for the record. Image previews are shown in record details on 'Log book' sheet.

  • GPS track
    If you have track for this flight, import it and select from the drop down list. Only tracks that are not attached to records shown in the list. Alternatively you can create record from a track.


For ground projects enter:

  • Date
    Date of the trip, default value is today, press arrow on the right to open calendar.

  • Duration
    Duration of the trip.

  • Length
    Total length of the trip.

  • Location
    Name of location.

  • Comments
    You notes for the record, comments are shown in record details on 'Log book' sheet.

  • Linked pictures
    Use 'Add picture','Delete' to attach and detach pictures for the record. Image previews are shown in record details on 'Log book' sheet.

  • GPS track
    If you have track for this flight, import it and select from the drop down list. Only tracks not attached to records are shown in the list. Alternatively you can create record from a track.
