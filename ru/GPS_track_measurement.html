Измерение трека

Трек-Альбом может определять следующие типы маршрутов:


Открытая дистанция:

T - точка взлета/старта
L - точка остановки/приземления
S - старт маршрута
F - финиш

S и F - точки с максимальной дистанцией




Маршрут через поворотную точку (ППМ):

S - точка взлета/старта, начало маршрута
F - финиш

a,b - отрезок маршрута
Дистанция: dist = a + b.

Требования:
любой отрезок / dist * 100 >= 20%
Угол h <= 90 градусов




Маршрут с возвращением:

S - точка взлета/старта, начало маршрута
2 - точка вне стартового цилиндра
F - финиш
a,b - отрезок маршрута

Дистанция: dist = a + b.




Треугольник ФАИ:

T - точка взлета/старта
L - точка остановки/приземления
S - старт маршрута
F - финиш

a,b,c - стороны треугольника
u - минимальная дистанция между начальным и конечным кусками трека

Дистанция: dist = a + b + c - u.

Требования:
любая сторона / (a + b + c) * 100 >= 28%
u / (a + b + c) * 100 <= 20%


В программе можно настроить минимальные дистанции для маршрутов, радиус стартового цилиндра, а так-же множитель очков за разные типы маршрута.
