Справка / Работа с записями / Создание и редактирование записи


Для создания записи на вкладке Журнал записей нажмите кнопку "Добавить", появится форма для ввода полей записи.

В зависимости от типа проекта на форме будут разные наборы полей.
Значения записей потом используются для подсчета статистики.

Обратите внимание, что есть поля где вводятся текстовые значения, и если нажать на стрелку справа, то появится список всех ранее введенных значений. Так же работает автоподстановка значений по первым буквам.


Для летных книжек заполняются:

  • Дата
    Дата полета, по умолчанью ставится текущей датой, нажмите на стролочку справа чтобы открыть удобный календарь выбора даты.

  • Крыло
    Название летательного аппарата, поле с автоподстановкой.

  • Наземная тренировка
    Установите эту галочку, если Вы не летали, а тренировались с крылом на земле. При этом будут автоматически установлены поля 'Тип полета', 'Метод старта', а так-же недоступны для редактирования некоторые поля.

  • Тип полета
    Например Маршрут, Динамик, Термодинамик...

  • Число полетов
    Если в этот день было несколько полетов то можно сгруппировать их в одну запись.

  • Общее время
    Суммарное время всех полетов.

  • Лучшее время
    Если полетов было несколько, то впишите время самого длительного из них.

  • Место
    Название летного места.

  • Метод старта
    Например 'Активная буксировка','Пассивная буксировка','Буксировка за самолетом', или оставьте пустым при старте с ног.

  • Горные полеты
    Отметьте эту галочку если полеты были в горах.

  • Событие
    Впишите здесь название поездки, например 'Маршрутный курс', 'Чемпионат России 2005', или оставьте поле пустым. В дальнейшем можно будет посмотреть статистику по событию.

  • Расстояние
    Заполняется для маршрутных полетов

  • Описание
    Ваши комментарии к записи. Комментарии будут показываться справа списка записей.

  • Ссылка на фото
    Используя кнопки 'Добавить фото' и 'Удалить фото' можно прикреплять файлы изображений к записи. Уменьшенное изображение будет показываться справа списка записей. (Внимание! Файлы изображений никуда не копируются)

  • GPS трек
    Если вы имеете трек полета, то выберите его из списка. Альтернативно Вы можете создать запись из трека.


Для наземных проектов заполняются:

  • Дата
    Дата поездки, по умолчанию ставится текущей датой, нажмите на стролочку справа чтобы открыть удобный календарь выбора даты.

  • Время
    Время поездки.

  • Расстояние
    Расстояние поездки.

  • Место
    Название места поездки.

  • Описание
    Ваши комментарии к записи. Комментарии будут показываться справа списка записей.

  • Ссылка на фото
    Используя кнопки 'Добавить фото' и 'Удалить фото' можно прикреплять файлs изображений к записи. Уменьшенное изображение будет показываться справа списка записей. (Внимание! Файлы изображений никуда не копируются)

  • GPS трек
    Если вы имеете трек поездки, то выберите его из списка. Альтернативно Вы можете создать запись из трека.

Если у Вас в какой-то день было несколько полетов/поездок, то вы можете сделайть либо несколько записей на одну дату, либо объеденить все полеты/поездки в одну запись, указав их количество и общее время.
