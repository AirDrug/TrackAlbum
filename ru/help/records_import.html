Справка / Работа с записями / Импорт записей из других приложений


Если Вы уже ведете записи в другой программе или электронной таблице,
то, вероятнее всего, вы захотите перенести данные в Трек-Альбом.

Трек-Альбом умеет импортировать данные из CSV файла (текстовый файл, где поля разделены точками с запятыми).

Если у вас записи ссылаются на треки, то прежде чем импортировать записи следует импортировать треки:
Выберите меню Файл/Импортировать трек, выберите файлы треков (можно выбрать несколько файлов за один раз, придерживая клавишу Ctrl).

CSV файл легче всего сделать в с помощью Excel, пример файла можете скачать тут: XLS файл.
После открытия файла сохраните его в формате CSV и импортируйте в Трек-Альбом (Внимание, закройте Excel перед импортом, иначе он заблокирует CSV файл).

Формат файла:

  • первая строка - комментарий
  • вторая строка - список полей для импорта
  • далее идут строки с данными
  • формат даты: такой как установлен в системе
  • формат времени: такой как установлен в системе
  • имена треков должны быть без расширения, т.е. так как они выглядят в списке треков

После того как CSV файл подготовлен:

  • выберите меню Файл/Импортировать записи
  • выберите CSV файл
  • просмотрите считанные данные (проблемные строчки помечены красным цветом, при наведении на них мышью всплывает подсказка, при этом кнопка подтверждения недоступна) и подтвердите импорт
